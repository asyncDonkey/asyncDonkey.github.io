About Me

While pursuing a Bachelor's degree (L-7) in Civil Engineering in Italy, foundational skills in Full-Stack Engineering are concurrently being developed. This path aims to bridge the analytical problem-solving inherent in engineering with the creative and practical applications of web technology.

An initial interest in software development was sparked by an Information Processing Systems course utilizing C during engineering studies. This quickly evolved into a broader exploration of programming, revealing particular engagement with automating processes, designing interfaces, managing data, and resolving complex logical challenges. A background in structured thinking, akin to language learning, provides a useful perspective for approaching programming languages.

Although early in this journey, the current focus is on mastering core web fundamentals – HTML, CSS, and JavaScript – primarily through hands-on learning via platforms like Codecademy and freeCodeCamp.

Key future goals include contributing to impactful open-source projects and collaborating within dynamic, proactive team settings. Opportunities for international work are also actively sought to gain diverse professional experiences and perspectives. Furthermore, there is a strong interest in identifying synergies between Civil Engineering knowledge and software development, potentially through creating tools for structural analysis, project visualization, or infrastructure data management.

Continuous learning and skill development in software engineering remain a core commitment.
