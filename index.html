Nuclear Energy Impact Simulator

A data-driven web application that simulates the economic impact of introducing nuclear power into Italy's energy mix. The backend, a Python script running on a daily cron job via GitHub Actions, fetches and processes generation/load data from the official ENTSO-E European API, performs a simulation using Pandas, and stores the results in Firebase Firestore.
